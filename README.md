# Proyecto Educativo: Flask con Swagger y Jinja2

## Descripción
Este proyecto tiene fines educativos y demuestra cómo crear una **aplicación web usando Python y Flask**, integrando las siguientes librerías:

- **Flask**: framework web ligero para Python.
- **Swagger**: documentación automática de APIs REST.
- **Jinja2**: motor de plantillas para generar contenido HTML dinámico.

El objetivo es aprender cómo construir APIs y páginas web dinámicas de manera simple y clara.

## Instalación

1. Clonar el repositorio:
- git clone https://github.com/alvarezmarlen/python_flask.git
2. Crear un entorno virtual:
- python -m venv venv
- source venv/bin/activate   # Linux/Mac
- venv\Scripts\activate      # Windows
3. Instalar las dependencias:
- pip install -r requirements.txt

## Uso
1. Ejecutar la aplicación:
- python app.py
2. Abrir tu navegador y visitar:
- http://localhost:5000
